YTKileroy Patreon Telegram Bots
@ListenForMeBot

Description

@ListenForMeBot is a telegram bot that enables users to use Google's Cloud Speech API to convert Telegram's audio messages into text with support of over 80 languages and variants!

If you wish to help support development of this bot, please consider one of the following:

Commands

/start Welcome message
/info Information about this bot
/help Help and commands
/chooselang Choose language and dialect
/chooselang lang-code Choose language and dialect from a language code (en-GB/gl-ES/etc.)
